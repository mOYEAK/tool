# 万能工具箱 V1 提审回归清单

## 使用方式

1. 提审前运行 `node scripts/preflight.js`，所有失败项必须清零。
2. 使用微信开发者工具稳定基础库编译，确认控制台无明显报错。
3. 上传体验版后，至少使用一台 Android 和一台 iPhone 完成下列回归。

## 必须通过

### 全局与首页

- [ ] 小程序首次打开正常，首页展示 10 个工具。
- [ ] 搜索“二维码”“签名”“PDF”“海报”“办公工具”结果正确。
- [ ] 所有工具入口均可进入，不出现页面未注册错误。
- [ ] 底部 Tab 仅展示“工具箱”和“我的”。
- [ ] 页面不存在乱码、缺字、按钮文字溢出或明显重叠。

### 工具闭环

- [ ] 九宫格切图：选择图片、生成九张图、批量保存成功。
- [ ] 长图拼接：选择至少两张图片、生成并保存成功。
- [ ] 二维码工具：文本生成、保存、从相册选择图片识别、复制结果成功。
- [ ] 手写签名：调整颜色和粗细、切换背景、保存结果与画板一致。
- [ ] 图片压缩：选择图片、调整质量、生成、保存成功。
- [ ] 带壳截图与壁纸：两种模式均可生成并保存。
- [ ] 图片转 PDF：图片缩略图正常、顺序调整正常、生成并预览成功。
- [ ] 名单抽取：输入名单、设置人数、抽取、复制、重新进入后恢复成功。
- [ ] 报价收据：填写项目、金额计算、生成、保存、草稿恢复成功。
- [ ] 活动海报：选择模板、填写内容、可选上传图片、生成、预览、保存成功。

### 权限与异常

- [ ] 首次保存图片时，相册权限申请流程正常。
- [ ] 拒绝相册权限后再次保存，能够引导打开设置。
- [ ] 用户取消选择图片时，不弹出误导性的失败提示。
- [ ] 断网后，所有本地工具仍可正常生成和保存。
- [ ] 超长图片、空输入、无效二维码等异常场景有明确提示且页面不崩溃。

### 我的与合规

- [ ] 打开工具后，“我的”页最近使用记录刷新。
- [ ] 成功生成或保存结果后，导出记录刷新。
- [ ] 清空本地记录后显示空状态。
- [ ] 隐私协议和关于我们页面可打开，文案无乱码。
- [ ] 未配置客服邮箱时，页面不展示示例邮箱。
- [ ] 页面明确说明当前未接入真实广告、支付和后端统计。

## 可接受警告

- [ ] 未配置客服邮箱，仅保留意见反馈和联系客服入口。
- [ ] 广告位保持安全占位，不加载真实广告组件。
- [ ] 支付、后端、OCR、AI 和第四阶段高风险功能保持关闭。
- [ ] 不同机型生成大图速度存在轻微差异，但无崩溃或卡死。
